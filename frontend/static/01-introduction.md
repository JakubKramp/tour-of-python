
## Introduction to Python

Before we dive into programming in python, lets briefly talk about the history of python. 
Python was created by [Guido van Rossum](/people/guido-van-rossum) a Dutch programmer in 1991. He was inspired by [ABC](https://en.wikipedia.org/wiki/ABC_(programming_language)), 
a programming language designed to be simple and its purpose was to teach programming. 
[Here](https://www.youtube.com/watch?v=J0Aq44Pze-w) is a brief history of why he decided to create Python by the man himself 

### Name

Python got his name from the British comedy series Monty Python's Flying Circus. 

### Why python

If you're wondering why should you should learn python out of the programming languages in the world, 
here are some reasons why I think Python is the best programming language for beginners (but not only for them).

- Simplicity. Python is a high level programming language, which mean it is quite close to natural language. It make it easy to pick up even for people for have never programmed before

- Popularity. Python is one of the most if not the most popular programming languages right now. That means that if you encounter an issue, there is high chance someone have already found a solution for your problem.

- Versatility. You can use python for pretty much anything. There are programming languages that are highly specialized. For example if you meet a Ruby Developer, there is 90% chance he writes web apps in [Ruby on Rails](https://rubyonrails.org/). With python you can do virtually anything, from scripting, or web apps, to data analysis and machine learning.
